◆ vec_group()

uint64_t vec_group ( int64_t *restrict vec,
uint8_t vec_len,
uint8_t group_bits )
inlinestatic

Group points together into groups of size 2^group_bits, modifying the vector to point to the new group.

If group_bits is divisible by vec_len, it's really quite simple. For example, 10 bits per group, 2 dimensions = 5 bits from each dimension, each group is 2^5 * 2^5. 10 bits per group, 5 dimensions = 2 bits from each dimension, each group is 2^2 * 2^2 * 2^2 * 2^2 * 2^2.

The real reason for this method beyond simple division is when group_bits is not divisible by vec_len. For example, for 3 dimensions and 10 bits it does 2^4 * 2^3 * 2^3.
